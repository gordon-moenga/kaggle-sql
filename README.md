# KAGGLE-SQL
This repository contains my solutions and notes from the SQL courses offered by Kaggle. The exercises document my progress in learning SQL for data analysis, beginning with foundational querying techniques and advancing to more complex analytical queries.

## Intro SQL

The Intro_SQL folder contains notebooks and exercises completed as part of Kaggle's Intro to SQL course.

### Topics covered include:
- Getting Started with SQL
- SELECT Statements
- WHERE Clauses
- GROUP BY, HAVING, and COUNT
- ORDER BY
- AS and Aliases
- Working with BigQuery
- Writing basic analytical queries

## Advanced SQL

The Advanced_SQL folder contains notebooks and exercises completed as part of Kaggle's Advanced SQL course.

### Topics covered include:
- JOIN Operations
- UNION
- WITH Clauses (Common Table Expressions)
- Analytic (Window) Functions
- Nested and Complex Queries
- Query Optimization
- Writing advanced analytical queries in BigQuery
